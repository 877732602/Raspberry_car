# 树莓派智能小车控制系统

[![Python3](https://img.shields.io/badge/python-3.6-green.svg?style=plastic)](https://www.python.org/)


## 简介
基于Tornado框架开发简洁树莓派智能小车web控制系统，实现手动控制方向，自动驾驶，试试摄像头监控，超声波测距，红外感应，自动寻迹，等汽车功能

## 启动
  
  python server.py
